# GuitarFX v0.1 alpha

Android + Oboeによるギター用リアルタイムUSBオーディオ・パススルーの最初のプロトタイプです。

## 現在の機能

- USBオーディオ入力から出力へのパススルー
- 入出力ゲイン
- 10 msスムージング付きミュート
- 入出力ピークメーター
- サンプルレート、frames per burst、バッファ、xRun表示
- Exclusiveで開けない場合はSharedへフォールバック

## 実行手順

1. Android Studioでこのフォルダーを開く。
2. SDK 35、NDK、CMake 3.22.1をSDK Managerから導入する。
3. Gradle Syncを実行する。
4. Android端末をUSBデバッグ接続して実機ビルドする。
5. アプリ起動前にUSBオーディオインターフェースを端末へ接続する。
6. オーディオインターフェースのDirect MonitorをOFFにする。
7. ヘッドホン音量を最小にしてからSTARTを押す。

## 重要

現段階ではAndroidが選択したデフォルトの入出力デバイスを使います。USB機器を明示選択する処理とUSB再接続処理は次の更新で追加します。

入力コールバックと出力コールバックを分離したリングバッファ方式ではなく、出力コールバックから入力をノンブロッキングreadする最小構成です。実機試験後、必要に応じてOboe FullDuplexStream相当の構造へ更新します。
